Detailed Description

Enum Tipo - definisce ogni tipo di tetramino con la sua forma di base e il colore.

Dettagli utili per feature future:

  • Le forme sono matrici 0/1; la rotazione viene gestita creando una matrice ruotata.
  • Per supportare forme più complesse o dati aggiuntivi (es. offset di spawn, SRS) potremmo estendere l'enum con parametri addizionali (es. pivot point, spawnOffset).

◆ getForma()

int[][] Tetris.Tipo.getForma ( )

Restituisce una copia profonda della matrice forma. Importante per evitare che modifiche temporanee sulla forma corrente modifichino la definizione del Tipo (immutabilità desiderabile).

Rischi/Edge-case:

  • La funzione clona le righe ma non valida la forma; se si aggiungono forme irregolari, implementare controlli.
